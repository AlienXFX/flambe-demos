# Horses demo

This is a port of the old singing horses Flash toy:
http://www.svt.se/hogafflahage/hogafflaHage_site/Kor/hestekor.html

All animation and scene layout is handled by [Flump], only a tiny bit of
code is required for the basic interactivity.

Try it out in [Flash], [HTML5] or [Android]. See also nickalie's
[CreateJS] version.

[Flash]: https://aduros.com/flambe/demos/horses/?flambe=flash
[HTML5]: https://aduros.com/flambe/demos/horses/?flambe=html
[Android]: https://aduros.com/flambe/demos/horses/main-android.apk
[Flump]: https://github.com/threerings/flump
[CreateJS]: https://github.com/nickalie/CreateJS-Haxe/tree/master/examples/SingingHorses
